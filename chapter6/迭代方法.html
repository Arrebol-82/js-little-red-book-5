迭代方法 (都是不会修改原数组)

every() 筛选元素

对数组中每个元素执行对应的传入函数,每一项返回true ,那这个方法才会返回true :: every() 对空数组返回 true

filter() 筛选元素

对数组中每个元素执行对应的传入函数,函数返回true的元素会组成新的数组返回来

forEach() 转换元素

对数组中每个元素执行对应的传入函数 没有返回值 相当于for循环

map() 转换元素

对数组中每个元素执行对应的传入函数,返回每次函数调用的结果,组成新数组

some()筛选元素

对数组中每个元素执行对应的传入函数,有一项返回true ,那这个方法才会返回true :: some() 对空数组返回 false

这些方法是只能用于数组,其他方法可以用于from 和 fo 转换为数组

      let numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1]
        因为有元素小于2 , 返回false 
      let everyResult = numbers.every((item, index, array) => { item > 2 })
      console.log(everyResult);
      
       因为数组有元素大于2, 返回true  
      let someResult = numbers.some((item, index, array) => item > 2)
      console.log(someResult);
      
       返回符合添件的新数组  
      let filterResult = numbers.filter((item, index, array) => item > 2)
      console.log(filterResult);
      
       对每个数组执行对应的函数  
      let mapResult = numbers.map((item, index, array) => item * 2)
      console.log(mapResult);
      
      
        这些写是没用的 
      let a = numbers.forEach((item, index, array) => item * 2)
      
       这样写  
      numbers.forEach((num, index, arr) => {
      console.log(`元素 ${num} 的索引是 ${index}，所在数组是 ${arr}`);
      });
