迭代器方法

因为数组是可迭代属性 , 所以会返回Array Iterator
keys() 是迭代器生成方法 需要通过 Array.from() 或循环将迭代器展开为数组。

像splice() 是数组操作方法 所以不用转换

      const a = ['foo', 'bar', 'baz', 'qux'];
      
      // 因为数组是可迭代属性 , 所以会返回Array Iterator { }
      // keys() 是迭代器生成方法 需要通过 Array.from() 或循环将迭代器展开为数组。
      console.log(a.keys());
      // 转换
      console.log(Array.from(a.keys()));
      
      // 像splice() 是数组操作方法 所以不用转换
  • keys() 返回数组 索引的迭代器
  • values() 返回数组 元素的迭代器
  • entries() 返回数组 索引/值对 （用数组包含）的迭代器

entries() 可以用数组结构更快拿到元素

      for (const [x, y] of a.entries()) {
      console.log(x);
      console.log(y);
      }
      
      0
      foo
      1
      bar
      2
      baz
      3
      qux
