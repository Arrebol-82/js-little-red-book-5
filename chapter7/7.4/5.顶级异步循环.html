           class Emitter {
      constructor(max) {
        this.max = max
        this.asyncId = 0
      }

      async *[Symbol.asyncIterator]() {
        while (this.max > this.asyncId) {
          yield new Promise((resolve) => resolve(this.asyncId++))
        }
      }
    }

    const emitter = new Emitter(5)

    其实本质都是一样的都是async环境 , 只是用法不一样
  • 本质: 顶级作用域不允许直接运行异步语法 ( 除了 type = 'module' 的script)
    一定要在async环境下运行
  • 手段:用 async 函数包裹，两种典型写法
    异步立即执行函数 :: 命名异步函数
  • 书中意图:用最简洁的例子讲清 “必须包裹”
    要想使用for-await-of 就要使用async环境
  • 我的写法:是属于在用于团队开发 要长期使用的 ( 至少豆包是这么说的 )
