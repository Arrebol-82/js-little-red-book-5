观察子列表变化

Mutationobserver可以观察到目标元素添加或删除子节点。这需要在MutationobserverInit中将 childList 属性设置为 true。

以下代码展示了添加节点:

      // 清除主体内容
      const test = document.getElementById('test')
          test.innerHTML = ''

      let observer = new MutationObserver(
        (mutationRecords) => console.log(mutationRecords)
      );

      observer.observe(test, { childList: true })

      // 添加一个 div
      test.appendChild(document.createElement('div'))

         type:"childList"

子节点重新排序虽然可以通过一个方法完成，但也会注册两个独立的变化,因为严格来说这涉及一次节点删除和一次节点再添加 

      // 清除主体内容

    const vessel = document.getElementById('vessel')
    vessel.innerHTML = ''

    let Observer = new MutationObserver(
      (mutationRecords) => console.log(mutationRecords)
    );

    // 先创建两个子节点
    vessel.appendChild(document.createElement('div'))
    vessel.appendChild(document.createElement('span'))

    Observer.observe(vessel, { childList: true });

    // 交换子节点的位置
    vessel.insertBefore(vessel.lastChild, vessel.firstChild)
    // insertBefore 将第一个元素(第一个节点) 插入到 第二个元素(第二个节点)前面
