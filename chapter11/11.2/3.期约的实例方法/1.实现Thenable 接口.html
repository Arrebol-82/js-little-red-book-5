实现Thenable接口

在 ECMAscript 暴露的异步结构中 , 只有自己定义 或者是 通过3Promise定义的实例 这些才会拥有 then( ) 方法。这个方法被认为实现了"henable"接口。 下面的例子展示了实现这一接口的最简单的类:

    class MyThenable {
      then() {

      }
    }

ECMAScript的Promise 类型实现了Thenable接口。这个简化的接口跟TypeScript 或其他包中的接口或类型定义不同，它们都设定了henable接口更具体的形式。

注意: 本章后面再介绍异步函数时还会再谈到Thenable 接口的用途和目的。
