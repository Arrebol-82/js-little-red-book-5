扩展操作符

扩展操作符可以让我们把两个对象以类似数组拼接的方式组合到一起，应用到内部对象的扩展操作 ， 将所有的可枚举的自有属性（包括符号）浅拷贝到外部对象

            const Foo = { a: 1 }
        const Bar = { b: 2 }

        const Foobar = { c: 3, ...Bar, ...Foo }
        console.log(Foobar); // {c: 3, b: 2, a: 1}

        const baz = { c: 4 }
        const foobarbaz = { ...Foo, ...Bar, c: 2, ...baz }
        console.log(foobarbaz) // a:1 , b:2 , c:4 ;

俩特性，看好了

  • 扩展操作符会严格按照扩展操作符的植入顺序，来植入对象
  • 第二就是跟平时写属性一样 ， 后面的属性会覆盖前面的属性
  • 来看看示例代码： 
                        const Foo = { a: 1 }
                    const Bar = { b: 2 }
            
                    const Foobar = { c: 3, ...Bar, ...Foo }
                    console.log(Foobar); // {c: 3, b: 2, a: 1}
            
                    const baz = { c: 4 }
                    const foobarbaz = { ...Foo, ...Bar, c: 2, ...baz }
                    console.log(foobarbaz) // a:1 , b:2 , c:4 ;

  • 浅拷贝

                        const Fo = { a: 1 }
                    const Ba = { b: 2, c: { d: 3 } }
                    const FoBa = {...Fo , ...Ba}

                    console.log(FoBa); // {a: 1, b: 2, c: {…}}
                    console.log(Ba.c === FoBa.c); // true
