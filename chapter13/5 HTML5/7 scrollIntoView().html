scrollIntoView()

DOM 规范中没有涉及如动页面中的某个区域。为填补这方面的缺失，不同浏览器实现了不同的控制滚动的方式。在所有这些专有方法中，HTMLS选择了标准化scro11Intoview()方法。

seroi11ntoview()方法存在于所有HTML元素上，可以滚动浏览器窗口或容器元素以便包含元素进入视口 。这个方法的参数如下:

  • alignToTop 是一个布尔值
  • true: 窗口滚动后元素的顶部与视口顶部对齐
  • false: 窗口滚动后元素的底部与视口底部对齐
  • scro11IntoViewOptions 是一个选项对象
  • behavior: 定义过渡动画，可取的值为"smooth"和"auto"，默认为"auto"。
  • block: 定义垂直方向的对齐，可取的值为“start"、"center"、"end"和"nearest"，默认为 'start'
  • inline: 定义水平方向的对齐，可取的值为"start"、"center"、"end"和"nearest"，默认为"nearest"。
  • nearest 如果当前元素在页面中的话就不懂 , 不在的话 , 靠左就往左边靠 , 靠右就往右边靠
  • 不传参数等同于 alignToTop 为 true

来看几个例子:

    function alignToTop() {
      document.forms[0].scrollIntoView(true)
    }


    function scrollIntoViewOptions() {
      document.forms[0].scrollIntoView({ behavior: 'smooth', block: "center", inline: "start" })
    }

这个方法可以用来在页面上发生某个事件时吸引用户的注意力。此外，把焦点设置到一个元素上也 会导致浏览器将元素滚动到可见位置。
